# BigData Aplicado Quiz

App de repaso para el examen de BigData Aplicado. 50 preguntas en 5 temas con repetición espaciada.

## Temas

- Almacenamiento y Arquitecturas
- Procesamiento Distribuido
- Calidad del Dato
- Observabilidad y DataOps
- Visualización y BI

## Uso

### Web
Abre la URL de Vercel en el navegador.

### Escritorio (Electron)
```bash
npm install
npm run dev
```

### Atajos de teclado
- `Espacio` / `Enter` — Ver respuesta
- `H` — Mostrar pista
- `1` — Correcto, `2` — Parcial, `3` — Fallé
- `←` / `→` — Navegar preguntas
